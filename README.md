# BARTOK

Peer-to-peer marketplace for **verifiable AI inference**, the "Uber of LLMs". Buyers chat and
pay per token from a real in-browser **Miden wallet**; the seller (João) serves each reply from
real models; every reply is **cryptographically verified** (which model ran, for how many
tokens) without exposing the API key, prompt, or answer; and sessions settle on **Miden
testnet** with an escrow split (seller paid, buyer refunded).

## What's here

See **`ARCHITECTURE.md`** for how the parts compose, and **`DEMO.md`** for the full runbook.

- **`ux-prototype/`**: the buyer app (Vite + vanilla JS + `@miden-sdk/miden-sdk` 0.15) with a
  real in-browser wallet that funds the escrow note, plus João's live seller dashboard
  (`seller.html`, SSE-fed) and the zero-dep bridge (`server.js`: sessions, tier routing,
  zkTLS pipeline, settlement).
- **`zktls-spike/`**: the proof lane. TLSNotary MPC-TLS over a real `openrouter.ai` call,
  selective disclosure of the labeled `model` + `usage` fields, a real notary keypair
  (`keys/`, generated by `openrouter_keygen`), and the **oracle** that verifies the proof +
  notary and derives `charge = tokens x tier price`. See `zktls-spike/RESULTS.md`.
- **`miden/`**: the on-chain half on protocol **v0.15**, standalone workspace (no external
  template checkout needed). `contracts/settlement-note/` is the BartokSettlement note
  (escrow in; charge as the note arg at consumption; two P2ID notes out), plus the testnet
  bins (`setup_accounts`, `escrow_params`, `fund_buyer`, `settle_session`, `smoke_escrow`)
  and the MockChain regression test.

## Quickstart

Prereqs once (details in `DEMO.md`): `zktls-spike/setup.sh`, an OpenRouter key in
`zktls-spike/.env`, `openrouter_keygen`, `setup_accounts`, `npm install` +
`npm run build:contracts` in `ux-prototype/`.

```bash
node ux-prototype/server.js        # the bridge; wait for [warm] ready
cd ux-prototype && npm run dev     # buyer UI  -> http://localhost:5173
# seller dashboard                 -> http://localhost:8787/seller.html
```

## Headless E2E
```bash
./stitch.sh          # both tiers notarized + oracle-verified, models asserted to differ,
                     # then a real escrow is settled on Miden testnet
./stitch.sh --mock   # MockChain settlement regression only (fast, offline)
```
